# Secciones

- Login
- Registro
- Usuarios logueados pueden:
  - Agregar nuevo producto
  - Modificar un producto
  - Eliminar o deshabilitar un producto
  - Agregar categorías
- Usuarios no registrados pueden:
  - Buscar información
  - Leer contenido
